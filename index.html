LLM Order Extraction Thesis

研究大型語言模型於異質訂單文件之關鍵資訊擷取與標準化， 建立從文件解析、prompt 設計到評估報表輸出的完整流程。
Research / AI
Python LLM Prompt Engineering JSON Schema OCR Evaluation
  • 處理異質訂單文件中欄位名稱變異、資訊分散、明細格式複雜與干擾資訊過多等問題。
  • 建立結構化抽取 schema 與評估流程，讓模型輸出可直接比較、分析與標準化。
  • Level 2 正式結果達 95.2% Document Accuracy，錯誤主要集中於日期格式判讀。
  • 進一步延伸至 Level 3 OCR / 影像實驗，分析視覺品質與抽取效能之關聯。
成果： 將原本非結構化、格式多變的訂單內容轉為標準化 JSON， 降低人工整理成本，並建立可重現的實驗與分析流程。
